# 小图上传项目

## 项目概述
这是一个通过 EO Pages 静态托管前端和 Node Functions 后端处理文件上传的项目。用户可以选择文件、填写伪装文件名和 Cookie，上传文件到闲鱼创作平台。

## 项目结构
- `index.html`：前端页面，提供文件上传功能。
- `node-functions/api/upload.js`：后端逻辑，处理文件上传，确保文件名唯一，并将文件转发到闲鱼接口。
- `README.md`：项目说明文档。

## 如何部署
1. 克隆该项目到本地。
2. 将项目上传至 EO Pages。
3. 配置环境变量 `ALLOWED_ORIGIN`，确保只允许指定来源调用上传 API。
4. 部署完成后，访问前端页面，进行文件上传。

## 使用说明
1. 打开前端页面。
2. 选择文件，填写伪装的文件名。
3. 输入闲鱼创作平台的 Cookie。
4. 点击上传按钮，文件将上传到闲鱼平台。

## 注意事项
- 文件上传大小限制：建议小于 5.5MB。
- 每次上传的文件名会自动加上 UUID，以避免文件名重复。
